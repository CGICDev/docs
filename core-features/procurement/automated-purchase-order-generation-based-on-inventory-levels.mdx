---
title: "Automated purchase order generation based on inventory levels"
description: "Automatically generate Purchase Orders (POs) when item stock falls below defined thresholds—streamlining replenishment and preventing stockouts in the CGIC App."
---

<Info>
  Prerequisites

  - Items must have **Reorder Level** and **Reorder Qty** set in their **Inventory** tab
  - Suppliers and default purchasing settings configured under each item’s **Purchasing** tab
  - Warehouses defined under `Stock > Warehouse `
</Info>

### 1.0 Set Reorder Levels on Items

1. Navigate to `Stock > Item > Item List.`  

2. Open the item you want to auto-reorder.  

3. Click the **Inventory** tab.  

4. Enter:  

   - **Reorder Level** (minimum on-hand quantity, e.g. 50 units)  

   - **Reorder Qty** (quantity to purchase when triggered, e.g. 200 units)  

5. Click **Save**.

### 1.1 Enable and Configure Auto-Reorder Rules

1. Go to `Stock > Auto Reorder > Auto Reorder List.`  

2. Click **New** to create a rule.  

3. Fill in:  

   - **Company** (e.g., Charisma Industrial)  

   - **Warehouse** to monitor (e.g., “Raw Material Store”)  

   - **Item Group** or specific **Items** (optional filter)  

   - **Reorder Based On**: choose **Reorder Level**  

   - **Purchase Order Lead Time** (days before expected delivery)  

4. Check **Enabled**.  

5. Click **Save**.

### 1.2 Run the Auto-Reorder Job

1. In **Auto Reorder List**, click **Run Now** (or schedule via **Scheduler Settings**).  

2. The system checks all enabled rules and items:  

   - If **Current Qty ≤ Reorder Level**, it generates a draft Purchase Order with the **Reorder Qty**.  

3. Navigate to `Buying > Purchase Order > Drafts` to review auto-generated POs.
