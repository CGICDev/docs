---

description: "The Raw Materials module in the CGIC App centralizes the management of all input materials—ensuring accurate tracking, procurement, usage monitoring, and cost control for seamless production support."
---

  <Card title="Add Material" icon="folder-plus" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/add-material">

  <Card title="Edit Material Details" icon="calendar-lines-pen" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/edit-material-details">

  </Card>
  <Card title="View Material" icon="bullseye-arrow" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/view-material">

  </Card>
  <Card title="Search Materials" icon="magnifying-glass-minus" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/view-material">

  </Card>
  <Card title="Delete Material" icon="bin-recycle" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/delete-material">

  </Card>
  <Card title="Categorize Materials" icon="group-arrows-rotate" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/categorize-materials">

  </Card>
  <Card title="Track Stock Levels" icon="chart-line" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/track-stock-levels">

  </Card>
  <Card title="Manage Suppliers" icon="people-pants" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/manage-raw-material-suppliers">

  </Card>
  <Card title="Track Purchases" icon="chart-mixed-up-circle-currency" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/track-purchases">

  </Card>
  <Card title="Set Reorder Points" icon="arrow-pointer" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/set-reorder-points">
    - Define minimum and maximum stock levels; automatically trigger purchase requisitions when levels drop below thresholds.
  </Card>
  <Card title="Monitor Usage" icon="monitor-waveform" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/monitor-usage">

  </Card>
  <Card title="Record Batches" icon="record-vinyl" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/record-batches">

  </Card>
  <Card title="Track Expiry Dates" icon="timeline" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/track-expiry-dates">

  </Card>
  <Card title="Track Movements" icon="timeline-arrow" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/track-movements">

  </Card>
  <Card title="Manage Returns" icon="house-person-return" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/manage-returns-to-suppliers">

  </Card>
  <Card title="Inventory Reports" icon="chart-user" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/inventory-reports">

  </Card>
  <Card title="Cost Tracking" icon="money-bills-simple" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/cost-tracking">

  </Card>
  <Card title="Record Spoilage" icon="chart-scatter-3d" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/record-spoilage">

  </Card>
  <Card title="Financial Reporting Data" icon="chart-mixed-up-circle-dollar" iconType="duotone" href="https://cgic.mintlify.app/core-features/raw-materials/financial-reporting-data">

  </Card>
