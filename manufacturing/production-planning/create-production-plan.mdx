  **What it does:**

  - **Schedule Production Runs**: Plan batches for CGIC products (e.g., _5,000 Valdana Bread loaves_) with start/end dates
  - **Auto-Calculate Material Needs**: Determine raw materials required based on BOMs and current stock
  - **Generate Execution Documents**: Create Work Orders for shop floors and Material Requests for procurement
  - **Optimize Resource Allocation**: Assign production to specific workstations (e.g., _Bottling Line 2_ for Ovita Waters)

### **1.0 Create New Production Plan**

1. Navigate to **Manufacturing \> Production Plan \> New**
2. Fill header details:
   - **Company**: _Charisma Industrial_
   - **Planning Type**:
     - _Material Request_ (to procure materials only)
     - _Work Order_ (to launch production)
     - _Both_
   - **Priority**: _High_ (for rush orders), _Medium_, _Low_

### **2.0 Add Products & Quantities**

1. In **Products Table**:

   | **Items**             | **Quantity** | **Planned Start Date** |
   | --------------------- | ------------ | ---------------------- |
   | Valdana Bread         | 5,000        | 2024-06-15             |
   | Ovita Waters 500ml    | 10,000       | 2024-06-16             |
   | Luca Engine Oil 5W-30 | 1,000        | 2024-06-17             |
2. For multi-level production:
   - ✓ Check **Include Sub-Assemblies** (e.g., _Pre-Mixed Oil_ for Luca Oil)

### **3.0 Calculate Material Requirements**

1. Click **Get Raw Materials**:
   - System auto-calculates needs based on BOMs and current stock
2. Review **Raw Materials Table**:

   | **Material** | **Required Qty** | **Available** | **Additional** |
   | :----------- | :--------------- | :------------ | :------------- |
   | Flour        | 2,500 kg         | 2,000 kg      | 500 kg         |
   | PET Bottles  | 10,000           | 8,000         | 2,000          |
   | Base Oil     | 950 L            | 1,200 L       | 0 L            |
3. Adjustments:
   - Add **Safety Stock** buffer (e.g., \+10% yeast for Valdana Bread)
   - Set **Scrap %** for loss-prone items (e.g., 5% Ovita caps)

### **4.0 Generate Execution Documents**

1. **Material Requests**:
   - Click **Create Material Request**
   - System auto-generates procurement requests for shortages
   - Example: _Request 500 kg flour from Supplier ABC_
2. **Work Orders**:
   - Click **Create Work Order**
   - Assign to workstations:
     - _Valdana Bread_ → Baking Line 3
     - _Ovita Waters_ → Filling Station 1
   - Set start/end times and operators

### **5.0 Track & Adjust**

1. Monitor progress via:
   - **Gantt Chart View**: Drag/drop timelines if delays occur
   - **Stock Level Alerts**: Real-time updates when materials arrive
2. Modify active plans:
   - **Add Urgent Orders**: Insert new rows (e.g., \+2,000 Onina Tissue packs)
   - **Reschedule**: Change dates via calendar icon
   - **Cancel**: Click **Hold** to pause a plan
